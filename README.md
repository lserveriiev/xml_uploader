# Upload and view xml files

##### Compile application:
`mvn package`

##### Run application:
`java -jar target/uploading-xml-0.1.0.jar` or `java -jar target/uploading-xml-0.1.0.jar --server.port=8080`


##### Navigate to browser: 
`http://127.0.0.1:8181`
